# Samples

This directory contains public, non-sensitive sample assets for onboarding and benchmarking.

- `benchmarks/invoices_small_structural_subset.json` is a curated structural subset manifest used by benchmark commands.
- `config.production.example.yaml` is a starter full config file you can copy outside the repo for private production runs.
- The main quickstart demo is generated on demand via `invoice-router demo --workspace .demo`, so the repository does not need to ship real invoice documents.
